# Welcome

This is the start page. Everything you see here lives in a single Markdown
file: `content/home.md`. Edit that file, commit it, and the change goes live.

Markdown lets you write **bold text**, *italics*, [links](https://example.com),
and lists without touching any HTML:

- Easy to read in plain text
- Easy to edit for non-developers
- No database, no admin login

## Getting around

Use the menu at the top to move between pages. Each menu entry is just
another Markdown file in the `content/` folder.
